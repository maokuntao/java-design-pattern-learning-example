## 责任链模式

### 定义：

> 责任链模式(chain of responsibility pattern):使多个对象都有机会处理请求，从而避免请求的发送者和接受者之间的耦合关系，将这个对象连成一条链，并沿着这条链传递该请求，直到有一个对象处理他为止。


### 适用范围：
> 
当想要让一个以上的对象有机会能够处理某个请求的时候，就是用责任链模式。
