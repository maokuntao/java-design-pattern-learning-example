## 代理模式

### 定义：

> 代理模式(proxy pattern) :为另一个对象提供一个替身或者是占位符，以控制对这个对象的访问。被代理的对象可以是远程的对象，创建开销大的对象或者是需要安全控制的对象。

##### 简单类图：
> 
![代理模式的简单类图](http://img.my.csdn.net/uploads/201205/08/1336453143_5004.png)

##### 使用场景：
>
- 远程代理控制访问远程对象
- 虚拟代理控制访问创建开销大的资源
- 保护代理基于权限控制对资源的访问
